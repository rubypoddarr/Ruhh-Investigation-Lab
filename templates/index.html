Ruhh Investigation Lab

Ruhh Investigation Lab is an AI-powered computer vision platform developed using Python, Flask, OpenCV, and YOLO. Users can upload any image and the system automatically detects, identifies, and labels objects present inside the image with intelligent visual analysis and real-time predictions.
