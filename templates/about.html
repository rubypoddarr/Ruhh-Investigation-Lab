Ruhh Investigation Lab is a Computer Vision based AI project developed as a BCA Final Year Project using Python, Flask, OpenCV, and YOLO. The system allows users to upload any image and automatically identifies objects present inside the image using Artificial Intelligence. Detected objects are highlighted with labels, confidence scores, and bounding boxes for visual understanding. The complete system works locally without relying on external APIs, making the project lightweight, fast, and fully open-source.

Python Flask OpenCV YOLO Computer Vision Artificial Intelligence

🔍 What Does This Project Do?

Ruhh Investigation Lab is an AI-based image analysis system. Users can upload any image and the platform automatically:

  • Detects objects using YOLO object detection
  • Identifies people, vehicles, animals, and everyday objects
  • Creates intelligent bounding boxes around detected items
  • Displays confidence scores for every prediction
  • Processes images fully offline without cloud APIs
  • Generates AI-powered visual understanding of the scene
Programming Language Python 3
Backend Framework Flask
Computer Vision Library OpenCV
Detection Model YOLO
Object Dataset COCO Dataset
Cloud APIs Not Required

🧠 How Does The AI System Work?

This project uses YOLO (You Only Look Once), one of the most powerful real-time object detection algorithms in Computer Vision. The model scans uploaded images and predicts objects using deep learning and neural network-based analysis.

1
Image Upload — User uploads any image file.
2
Image Processing — OpenCV prepares the image for AI analysis.
3
YOLO Detection — AI identifies visible objects inside the image.
4
Bounding Boxes — Detected objects are highlighted visually.
5
Prediction Results — Final image with AI detections is generated.
