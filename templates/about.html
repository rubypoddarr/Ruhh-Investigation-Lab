Ruhh Investigation Lab is my final year BCA project — a practical demonstration of applied Computer Vision using Python, Flask, and the YOLOv3 object detection model. The system works completely offline without any external APIs or paid services. Every image you upload is processed locally, detected with a neural network trained on 80 everyday object categories, and returned with a structured investigation report.

Python Flask OpenCV Machine Learning Computer Vision

🔭 What is This Project?

Ruhh Investigation Lab is an image analysis web application. Upload any photograph and the system will:

  • Detect objects using YOLOv3 (80 COCO classes)
  • Draw colour-coded bounding boxes with confidence scores
  • Classify the scene as Indoor / Outdoor / Mixed
  • Generate structured investigation-style observations
LanguagePython 3.11
FrameworkFlask 3
CV LibraryOpenCV DNN
ModelYOLOv3-tiny
DatasetCOCO (80 classes)
API KeysNone required

🧠 How Does Computer Vision Work?

Computer Vision teaches machines to interpret images the way humans do. This project uses YOLOv3 (You Only Look Once), a convolutional neural network that divides an image into a grid and predicts bounding boxes and class probabilities in a single forward pass.

1
Image Input — User uploads PNG/JPG/etc.
2
Pre-processing — Resized to 416×416, normalised to [0,1]
3
YOLO Forward Pass — Network predicts boxes & class scores
4
NMS Filtering — Non-Maximum Suppression removes duplicates
5
Report Generation — Scene classified, observations written
